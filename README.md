# Cybersecurity-Internship-Projects
This is a compendium of cybersecurity and information security projects completed during the internship demonstrating practical cybersecurity skills acquired covering governance, risk management, awareness, and technical assessments.

- `01_information_security_policies/` — policy templates and guidance
- `02_governance_frameworks/` — comparison of NIST, ISO, CIS
- `03_risk_management/` — risk register examples
- `04_cisv8_security_assessment/` — CIS Controls v8 mapping
- `05_third_party_risk_management/` — vendor risk questionnaire
- `06_cybersecurity_awareness_training/` — training materials
- `07_incident_response_simulation/` — tabletop exercise report
- `08_pen_test_vulnerability_assessment/` — sample findings
